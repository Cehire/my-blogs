Python的语言中的变量不需要显式指出数据类型的声明

a: int = 10
b = 10

Python能够自动推断出变量的类型

# 数字类型

表示数字或者数值的数据类型就是数字类型

  • 整形（int）：Python3之后没有长整型（long）只有int
  • 浮点型（float）
  • 复数类型（complex）
  • 布尔类型（bool）

# 整型

整型的表示形式：

  • 二进制：以0b或者0B开头
  • 八进制：以0o或者0O开头
  • 十进制：默认
  • 十六进制：以0x或0X开头

整型不同类型的转换

  1. bin()：十进制转换为二进制，输出形式是字符串
  2. oct()：十进制转换为八进制，输出形式是字符串
  3. hex()：十进制转换为十六进制，输出形式是字符串
  4. int()：接收一个符合整型规范的字符串，并将字符串转换为整型

Python中，当多个变量的值相同时，这个值只在内存中保存一次，多个变量都指向同一内存。数字是Python中的不可变对象，当要修改它的值时，并不是真的修改内存中变量的值，而是**把新的值放入内存中，然后修改变量，使得变量指向新值的内存地址。**浮点数、复数等其他数字类型及其他类型的不可变对象具有同样的特点

Python具有自动管理内存功能，如果一个值没有任何变量指向，则Python会自动将其删除，可以显式地使用del命令删除不再使用的值。

# 浮点数

浮点数就是数学中的小数

  • Python中的浮点型一般以十进制表示，由整数和小数部分（可以是0）组成
  • 对于很大或者很小的浮点型可以使用科学计数法表示 （1.2e-2：1.2乘以10的-2次方）
  • Python中的浮点型是双精度的，每个浮点型的数据占8个字节

# 复数

  • 复数由“实部”和“虚部”组成
  • 实数部分和虚数部分都是浮点型
  • 虚数部分后面必须要有j或者J

Python中表示复数有两种方法

  1. a+bj
  2. complex(a,b)

其中a表示实部，b表示虚部

# 布尔类型

  • Python中的布尔类型只有True和False两个取值
  • True对应整数1，False对应整数0
  • 常用的布尔运算包括and、or、not三种

等同于False的值

  • None
  • False
  • 任何为0的数字类型，如0、0.0、0j
  • 任何空序列，如""、()、[]
  • 空字典{}
  • 用户定义的类实例中定义了__bool__()或者__len__()方法，并且该方法的返回0或者False

等同于True的值

  • 非0整数值
  • 非空字符串

短路法则

Python中的and和or运算中具有短路效果

# 字符串类型

在Python中，字符串有三种表示方式

  • 单引号
  • 双引号
  • 三重引号

# 单引号、双引号

在Python中，单引号和双引号表示的字符串是等价的，没有本质的区别，可以在一种引号中嵌套另一种引号。

# 三重引号

三重引号可以三个单引号，也可以三个双引号。这种方式表示的字符串也叫做块字符串。三重引号是以三个同一种类型的引号开始，并以三个相同引号结束的字符串表示方式。

# 转义字符

# raw字符串

# 元组的优势
  • 可以使函数返回多个值
  • 可以使程序运行性能提升
  • 一般来说，创建元组类型tuple的变量比列表类型list要快，而且占用更小的存储空间

# set（集合）

创建集合对象：变量名 = {元素1,元素2,...,元素n}

集合的特性：

  • 无序性：元素之间没有确定的顺序
  • 互异性：不会出现重复的元素
  • 确定性：元素和集合只有属于或者不属于的关系

注意：创建空集合要使用set();，{}创建的是空字典
