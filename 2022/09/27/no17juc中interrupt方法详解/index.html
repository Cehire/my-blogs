JUC中interrupt方法详解
  • 2022-9-27
  • 2022-9-27
  • Java多线程

首先，一个线程不应该由其他线程来强制中断或者停止，而是应该由线程自行停止。所以Thread.stop，Thread.stop，Thread.suspend，Thread.resume都废弃了。而Thread.interrupt的作用也不是中断线程，而是通知线程该中断了，具体的中断逻辑（或者是不中断）应该由被通知的线程自行处理。

# 中断的相关方法

  • public void interrupt()：将调用者线程的中断状态设置为true
  • public boolean isInterrupted()：判断调用者线程的中断状态
  • public static boolean interrupted()：是Thread类的静态方法，只能由Thread调用，它会返回当前线程的中断状态，并将当前线程的中断状态置为false

# 调用interrupt()

  • 如果线程处于被阻塞状态，那么线程将立即退出被阻塞状态，同时抛出InterruptedException异常
  • 如果线程处于非阻塞状态，那么将会将该线程的中断标志设置为true
  • 注意：在线程被同步锁或者是IO操作锁阻塞是，调用interrupt方法时线程无法去检查中断状态或者抛出InterruptedException（interrupt方法被阻塞）
  • 线程根据自己的逻辑判断中断位并进行操作
package cn.cehire;

import java.util.concurrent.TimeUnit;

public class Demo3 {
    //对于interrupt方面的详细分析
    public static void main(String[] args) throws InterruptedException {
        //Thread的静态方法interrupted能判断isInterrupted标志位，同时将标志位置为false
        //线程的成员方法的isInterrupted也能判断isInterrupted标志位，但是不会更高isInterrupted标志位
        //线程的interrupt方法会使得该线程的isInterrupted标志位由false改成true
        //如果一个线程在睡眠时调用interrupt方法，则会抛出InterruptedException异常，同时isInterrupted置为false
        Thread t1 = new Thread(){
            @Override
            public void run() {
                System.out.println(this.getName() + "运行中");
                System.out.println(this.getName() + "睡眠5s");
                try {
                    TimeUnit.SECONDS.sleep(5);
                } catch (InterruptedException e) {
                    System.out.println("由于睡眠时中断而抛出异常");
                    e.printStackTrace();
                }
            }
        };
//        t1.start();
//        t1.interrupt();
        //同样地，在isInterrupted为true的线程执行sleep方法时会将isInterrupted置为false，同时抛出一个异常
        //其他阻塞状态的情况也同此
        Thread t2 = new Thread("t2"){
            @Override
            public void run() {
                System.out.println(this.getName() + "运行中");
                System.out.println(this.getName()+":"+this.isInterrupted());
                this.interrupt();
                System.out.println(this.getName()+":"+this.isInterrupted());
                try {
                    TimeUnit.SECONDS.sleep(2);
                } catch (InterruptedException e) {
                    System.out.println("抛出异常");
                    System.out.println(this.getName()+":"+this.isInterrupted());
                    e.printStackTrace();
                }
            }
        };
//        t2.start();
        Thread t3 = new Thread("t3"){
            @Override
            public void run() {
                System.out.println(this.getName() + "运行中");
                if(Thread.interrupted()){
                    System.out.println("执行中断逻辑");
                }
            }
        };
//        t3.start();
//        t3.interrupt();
    }
}
