# 切换视角

  1. 按住鼠标中键拖动鼠标，可以旋转视角
  2. 按住Shift+鼠标中键，然后拖动鼠标，可以平移视角
  3. 滚动鼠标中键缩放

通过这三者的配合，可以实现绝大部分视角移动需求

按住==`==键（TAB键上面那个），会弹出复选框，可以选择正视图等特定视角

# 选择

  1. 单点选择：快捷键W
  2. 框选：长按左键拖动，框柱的对象会被选中，快捷键B
  3. 刷选：长按左键拖动，经过的对象会被选中，快捷键C
  4. 套索选择：类似于框选，快捷键L

# 游标

==游标可以设置新建对象的位置==，选择游标后点击左键可以设置游标的位置，shift+c可以将游标的位置设置在原点(0,0,0)

# 移动

选择物体后使用快捷键G来简略地移动对象；选择物体后再切换到移动会出现坐标轴和对应平面方便进行移动。

//使用快捷键进行精准的移动（单位m）
//1. 选择物体
//2. G+X/Y/Z+移动的距离（X/Y/Z表示按照X/Y/Z轴移动）
//3. 回车确认

# 旋转

选择物体后使用快捷键R来简略地旋转对象；选择物体后再切换到旋转会出现旋转轴方便进行旋转

//使用快捷键进行精准的移动（单位°）
//1. 选择物体
//2. R+X/Y/Z+旋转的角度（X/Y/Z表示按照X/Y/Z轴旋转）
//3. 回车确认

# 缩放

选择物体后使用快捷键S来简略地缩放对象；选择物体后再切换到缩放会出现缩放轴方便进行缩放

//使用快捷键进行精准的移动（单位倍数）
//1. 选择物体
//2. R+X/Y/Z+缩放的角度（X/Y/Z表示按照X/Y/Z轴缩放）
//3. 回车确认

Alt+S为法相缩放，法相缩放只会对其法线方向进行缩放，其他方向不做改变

# 按TAB键进入编辑界面

数字键1、2、3切换点、线、面模式

==Z==键出现模式选框，可以选择线框、实体等模式
